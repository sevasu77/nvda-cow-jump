
# 🐄 Hyper Cow Jump – Physics Hunger Survival Game

> **"What if the stock market became a physical law of nature?"**

## 💡 Concept
This project is an **experimental data visualization and educational simulation**. 
It explores how abstract financial data (market volatility) can be translated into physical forces (gravity and jump mechanics) within a virtual environment.

### 🛡️ Disclaimer (Important)
- **Not Financial Advice:** All data and information presented are for educational and entertainment purposes only.
- **Educational Experiment:** This is an independent art piece exploring data sonification and physics. It is not affiliated with, endorsed by, or representing NVIDIA, ASML, or any other real-world entity.
- **Data Accuracy:** Market data is provided via `yfinance`. It is not real-time and may be delayed.
- **Risk:** The developer is not responsible for any financial losses or decisions made based on this experimental software.

💡 Concept: The Semiconductor Meta-Causality
"Why is a cow powered by NVIDIA (NVDA) carrying an ASML financial report on its back?"

Because in the modern semiconductor ecosystem, there is a brutal truth: "If ASML trips, NVIDIA cannot jump." In this game, the cow's jumping power (propulsion) is entirely dependent on N



VIDIA's stock price fluctuations. However, the technical foundation supporting its feet is concentrated in the ASML "chip" (QR code) on its back. This represents the delicate balance of the global supply chain.

🕹️ Game Mechanics
Real-time Gravity: When NVIDIA's stock rises, gravity decreases. When it crashes, the cow is slammed to the ground by the weight of market despair.

Dynamic MOO-Synth: A "Moo" sound synthesized from scratch via JavaScript. It becomes light and melodic when the market is positive, but turns into a deep, haunting bass from the depths of hell when the market is negative.

Analysis Mode: The market never stops, but this game does. Press [P] to pause. The microchip (QR code) on the cow's back scales up, allowing players to scan and read ASML's IR reports to escape from the reality of price action.

🛠️ Tech Stack
Frontend: HTML5 Canvas, JavaScript (Web Audio API for Synthesizer)

Backend: Python, Streamlit

Data Source: yfinance (Real-time stock data)

Library: qrcode.js

## 🇯🇵 プロジェクトの理念と技術的背景（日本語）

### 🌌 コンセプト: 「市場は自然界の物理法則になり得るか？」
このプロジェクトは、単なるゲームではなく、**「データ駆動型のインタラクティブ・シミュレーション」**です。
株価の変動（ボラティリティ）という抽象的な数値を、仮想空間内の重力や跳躍力といった物理法則に変換することで、投資家が直面する市場の「重圧」や「不条理」を視覚・聴覚的に体験することを目的としています。
